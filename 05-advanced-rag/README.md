# 05 — Advanced RAG

Patterns that move a RAG system from "demo" to "production":

- **Query rewriting** — rephrasing the user's question for better retrieval
- **Hybrid search** — combining dense and sparse retrieval
- **Re-ranking** — a second pass with a cross-encoder to sharpen results
- **Contextual compression** — trimming retrieved chunks before handing them to the LLM
- **Multi-query / fusion retrieval** — issuing several retrieval queries and merging results
